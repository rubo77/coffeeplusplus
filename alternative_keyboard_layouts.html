The problem with the Dvorak layout is that it is not very handy to use on a laptop with a touchpad and that the function keys like Shift, Enter... are too far to reach. The Dvorak layout is optimized for handicapped people, that really have only one hand.

To combine the advantages of having one hand free and using both hands to reach maximum typing speed the ENTI-Key Layout is optimized:
